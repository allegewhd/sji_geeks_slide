What I did in GREE

Software engineers in GREE

What we do

develop and operate mobile social game services running on GREE platform

  • analyze history log and decide new game function with game directors
  • update game resource files(animations, images and sounds) for new function
  • develop, QA support(set up QA server, etc.), code review
  • staging deploy and production deploy
  • server status monitor and support infrastructure team to resolve hardware issues
  • some server backup stuff
  • resolve issues from custom support team
  • update iOS and Android client application

  • high level of personal management

Software engineers in GREE

Good things and bad things

Good parts

  • grow up quickly
  • all information are opened to you
  • less office politics
  • less distraction, you can focus on code
  • take risk and do not fear to fail

Bad parts

  • you need careful health management
  • redundant information make you nervous
  • annoying gree alert mail in the middle of night

Know how to understand, communicate with, and collaborate with other people

How to be a geek

Some stuff from GREE engineer advices for incomer

  • Love code and try to ship great software
  • Get into open source world
  • Have fun with effort
  • Lose the ego
  • Pursue the bottom matter, not just chitchat
  • Do more practice to be professional
  • Be in friendly rivary with other geeks
  • Automate some task you do every day
  • Learn new computer language
  • Start open source project on github or bitbucket.
